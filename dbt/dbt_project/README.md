Welcome to your new dbt project!

### Using the starter project

Try running the following commands:
- cd dbt/taxi_dbt
- dbt deps --profiles-dir ../dbt_profiles --target dev
- dbt deps --profiles-dir ../dbt_profiles --target dev
- dbt run --select tag:gold --profiles-dir ../dbt_profiles --target dev
- dbt test
